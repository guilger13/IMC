# 🧮Calculadora IMC

Projeto desenvolvido com **HTML, CSS e JS** com o objetivo de elaborar uma UI que calcula o IMC.
O projeto inclui CSS e manipulação de campos através do JS.

## 🚀Tecnologias Utilizadas
- HTML5
- CSS3
- Javascript (Vanilla JS)
